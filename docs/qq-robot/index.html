QQ机器人(智障宽宽)

功能

  • 小作文+关键字:狗屁不通小作文

  • gpt+问题:向chatGPT提问(无上下文关联)

  • talkgpt+内容:与chatGPT聊天(内容更日常，有上下文)

  • 绘画+内容:openAI绘画

  • 天气+城市:获取天气信息

  • 加密+内容:兽语加密

  • 解密+内容:兽语解密

  • 百科+内容:搜索百度百科

  • 翻译+内容:QQ翻译

  • 地图+地点:查询地图

  • 交通+地点:查询交通情况

  • 计算+内容:运算表达式

  • IP+地址:查询IP地域

  • 新闻:实时新闻

  • up搜索+关键字:搜索B站UP主

  • up信息+关键字:展示B站UP主信息

  • 一言

  • 歌曲相关

    • 点歌+歌曲名[+平台识别码]:普通点歌

      • 支持网易云,QQ音乐,酷我音乐

      • e.g.点歌明天你好;

    • 破解+歌曲名[+平台识别码]:破解歌曲下载地址

      • 默认QQ音乐

      • e.g.破解明天你好

    • 歌曲+歌曲名[+参数][+平台识别码]：歌曲高级搜索

      • e.g.歌曲明天你好$

    • 平台识别码

      • 不加:默认(破解默认QQ音乐,其他默认网易云)

      • 分号:QQ音乐

      • $号:酷我音乐

  • 早上好/早啊:早安问候

  • 抱抱:来自宽宽的抱抱

  • 你好:信息面板

  • 功能:本功能面板

  • 赞助:赞助作者

  • 群事件:

    • #接龙

    • #投票

    • #抽奖

  • 测试中:

    • base64[内容]:base64编码

    • 二维码[内容]:制作二维码

  • 群聊管理员:

    • 黑名单(单独语法)群黑名单

    • 欢迎(单独语法)设置欢迎语

  • 已停用:

    • 疫情[城市]:疫情查询

  • 意见反馈:

    • 反馈[内容]

示例

  • @智障宽宽 点歌明天会更好

  • @智障宽宽 地图重庆

  • ETC.

使用方法

  • 加QQ好友2439973472

  • 可以拉入群聊

  • 需要管理员同意，所以申请后可能需要耐心等待一会

注意事项

  1. 对于QQ来说,@是一个特殊格式不能复制

  2. 单个用户100s限制20次，群聊100s限制50次

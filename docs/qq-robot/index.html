QQ机器人(智障宽宽)

功能

  • 天气[城市]:获取天气信息

  • 点歌[歌名]:网易云音乐分享

  • 点歌[歌名];:QQ音乐分享

  • 破解[歌名]:QQ音乐下载破解

  • 加密[内容]:兽语加密

  • 解密[内容]:兽语解密

  • 百科[内容]:搜索百度百科

  • 翻译[内容]:QQ翻译

  • 地图[地点]:查询地图

  • 交通[地点]:查询交通情况

  • 早上好/早啊:早安问候

  • 抱抱:来自宽宽的抱抱

  • 你好:信息面板

  • 功能:本功能面板

  • 赞助:赞助作者

  • 群事件:

  • #接龙

  • 投票

  • 抽奖

  • 测试中:

  • 计算[内容]:运算表达式

  • IP[地址]:查询IP地域

  • 歌曲[歌曲名]:高级查询

  • 群聊管理员:

  • 黑名单(单独语法)群黑名单

  • 欢迎(单独语法)设置欢迎语

  • 需要特殊权限:

  • $sudo:管理员指令

  • gpt[问题]:chatGPT提问

  • 绘画[内容]:openAI绘画

  • 已停用:

  • 疫情[城市]:疫情查询

  • 意见反馈:

  • 反馈[内容]

示例

  • @智障宽宽 点歌明天会更好

  • @智障宽宽 地图重庆

  • ETC.

注意事项

  1. 对于QQ来说,@是一个特殊格式不能复制

  2. 单个用户100s限制20次，群聊100s限制50次

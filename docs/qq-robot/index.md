# QQ机器人(智障宽宽)

## 功能

- openAI相关
  
     - gpt+内容:GPT-3原版提问
     - talkgpt+内容:聊天优化的GPT-3提问(带上下文关联)
     - codegpt+内容:代码补全(模型code-davinci-002)
     - chatgpt+内容:与chatGPT聊天(原生支持)
     - 在任意关键字前加open答案会更开放、有创造性、不合常理(temperature为0.9)
     - 绘画+内容:openAI绘画

- 天气+城市:获取天气信息

- 地图+地点:查询地图

- 交通+地点:查询交通情况

- 计算+内容:运算表达式

- IP+地址:查询IP地域

- 新闻:实时新闻

- up搜索+关键字:搜索B站UP主

- up信息+关键字:展示B站UP主信息

- gitee+内容:搜索Gitee仓库

- github+内容:搜索Github仓库

- 一言

- 歌曲相关
  
     - 点歌+歌曲名[+平台识别码]:普通点歌
       
          - 支持网易云,酷我音乐
          - e.g.点歌明天你好;
  
     - 破解+歌曲名[+平台识别码]:破解歌曲下载地址
       
          - e.g.破解明天你好$
          - e.g.破解明天你好
  
     - 歌曲+歌曲名[+参数][+平台识别码]：歌曲高级搜索
       
          - e.g.歌曲明天你好
          - e.g.歌曲明天你好$
  
     - 平台识别码
       
          - 不加:默认(破解默认QQ音乐,其他默认网易云)
          - 分号:QQ音乐
          - $号:酷我音乐

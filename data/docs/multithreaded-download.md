# 多线程下载器(autoDownload)

## 安装

```bash
pip install autoDownload
```

## 使用

### import

```python
import autoDownload

autoDownload.AutoDownload(URL,file,key=value...).start()
```

### 命令行

```bash
auto-download URL
```

其他参数详见

```bash
auto-download -h
```
